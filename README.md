## 🎯 Sobre este Repositório

Este projeto é um **fork** da curadoria de [Data Science da OSSU](https://github.com/ossu/data-science). O objetivo desta versão mantida por **Ariel Shlomoh** é:

1.  **Tradução e Adaptação:** Facilitar o acesso para a comunidade lusófona.
2.  **Interface Moderna:** Refatoração completa utilizando **MkDocs** com o tema **Material (Neon Dark)** para uma melhor experiência de leitura.
3.  **Acompanhamento:** Servir como base para o meu tracker pessoal de progresso na trilha de Ciência de Dados.

## 📚 Currículo Resumido

O currículo original da OSSU é focado em fornecer uma educação de nível universitário gratuita em Ciência de Dados, utilizando cursos das melhores universidades do mundo (MIT, Harvard, Stanford, etc).

### Pilares da Trilha:
*   **Introdução:** Python, Computação e Pensamento Computacional.
*   **Fundamentos Matemáticos:** Cálculo, Álgebra Linear e Estatística/Probabilidade.
*   **Engenharia de Dados:** Bancos de Dados, SQL e Data Warehousing.
*   **Análise & Modelagem:** Ferramentas de Data Science e Machine Learning.
