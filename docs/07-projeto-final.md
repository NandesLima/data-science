# 🎯 Projeto Final e Portfolio

Um cientista de dados é contratado pelo que ele consegue fazer, não apenas pelo que ele sabe. Seu projeto final deve demonstrar domínio de todas as ferramentas e uma visão clara de negócio.

### Objetivos:
*   Identificar um problema real que possa ser resolvido com dados.
*   Coletar, limpar e analisar o dataset necessário.
*   Modelar a solução e documentar o processo no GitHub.
*   Apresentar os resultados de forma clara e orientada a decisões.
